# Sudoku Generator

Mit diesem Projekt wird demonstriert, wie Sudoku Rätsel mit Hilfe von Quantenrechnern gelöst werden können.
Dazu wird abhängig vom zu lösenden Sudoku ein Quantenschaltkreis mit IBM Qiskit generiert.
Auf dieser Website wird die Funktionsweise der Anwendung Schritt für Schritt beschrieben.

## Motivation
Es ist bekannt, dass das Lösen von Sudoku Rätseln ein NP vollständiges Problem ist[^1][^2].
Das bedeutet, dass es nicht möglich ist, in polynomialer Zeit eine Lösung zu finden.
Die bestmögliche bekannte Laufzeitkomplexität liegt somit bei $O(n)$.

Mithilfe von Quantencomputern und unter Verwendung des Grover Algorithmus kann das Lösen
von Sudoku Rätseln in $O(\sqrt{n})$ gelöst werden. 
Zum jetzigen Zeitpunt ist das Lösen solcher Rätsel mit echten Quantenrechnern nicht möglich,
da die heutige Quantenhardware zu wenige Qubits bietet und noch zu fehleranfällig ist.

Trotzdem sind für Softwareentwickler schon heute Kenntnisse über die Funktionsweise von Quantencomputern und das Entwickeln von
Quantenalgorithmen von Interesse um für die Zukunft gerüstet zu sein.

## Die Anwendung
Es wurde eine Anwendung entwickelt, welche unter Angabe eines Sudoku Rätsels
einen Quantenschaltkreis zur Lösung des Rätsels generiert.

Der erstellte Quantenschaltkreis kann anschließend auf einem Simulator ausgeführt,
oder im QASM[^3] Format exportiert werden. Der QASM Code kann verwendet werden, um den
Schaltkreis auf echter Quantenhardware auszuführen zu lassen.

Mehr Informationen zur Anwendung inklusive einer Live-Demonstration ist im [Kapitel Demo Anwendung](/quantenalgorithmus/uebersicht/demo-anwendung) zu fiden. 

## Der Quantenalgorithmus

Der entwickelte Quantenalgorithmus setzt den [Grover Algorithmus](/basiswissen/grover) ein.
Unsere Implementierung basiert auf einen Ansatz, der im Rahmen des Hackathons "Qiskit Camp Asia 2019"
vom Github User [yoooopeeee](https://github.com/yoooopeeee) entwickelt wurde.
Unsere Erweiterung bewirkt, dass der Algorithmus nicht wie zuvor nur gewisse 4x4 Sudokus, sondern
jegliche Sudoku Rätsel mit beliebiger Größe lösen kann[^4].

Der genaue Aufbau, sowie die einzelnen Bestandteile des Quantenalgorithmus ist im [Kapitel Quantenalgorithmus](/quantenalgorithmus/uebersicht/bestandteile) beschrieben.
